##### Display Flex parent level properties

1. flex-flow
This contains flex-direction & flex-wrap.

Example:
```bash
flex-flow: row wrap;
```
flex-direction functions by Main Axis vs. cross axis concept
```bash
# flex-direction: row
---> main axis
|
|
cross axis
```
&
```bash
# flex-direction: row-reverse
main axis <-----|
                |
                |
                cross axis
```
